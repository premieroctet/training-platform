#### Déclarer des types

Le typage se fait juste après la déclaration d'une variable

```typescript
const username: string = "John Doe";
```

---

<SlideHeader slideTitle="Les différents types" />

#### Types primaires

String, number, boolean

```ts
const username: string;
const old: number;
const isActive: boolean;
const skills: string[];
```

---

<SlideHeader slideTitle="Les différents types" />

#### Types par référence

Array, tuple

```typescript
// Deux façon de typer un array
const roles: Array<string>;
const roles: string[]; // <-- Méthode la plus utilisée

// Tuples
const settings: [string, string, number];
```
