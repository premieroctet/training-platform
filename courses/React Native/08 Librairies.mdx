#### Nombreuses librairies compatibles mais...

- Nous ne sommes pas dans un navigateur, les librairies faisant références à
  "window" ou "document" ne fonctionnent pas.
- Idem pour les librairies utilisant nodeJS
- Attention au choix :
  - parfois plusieurs librairies font la même chose
  - beaucoup ne gèrent qu'une seule plateforme
  - projets abandonnées ou peu maintenu

#### Quelques exemples de librairies utiles

- [react-intl](https://github.com/formatjs/formatjs): Internationalisation
- [Lodash](http://underscorejs.org/) : utilitai res (tri, filtre, throttling...)
- [ImmerJS](https://github.com/immerjs/immer) : simplifie les données immutables
- [Redux](http://redux.js.org/) : architecture
- [Rematch](https://github.com/rematch/rematch) : Helpers Redux pour éviter du
  boilerplate
- [React hook form](https://react-hook-form.com/) /
  [Formik](https://github.com/jaredpalmer/formik) /
  [React Final Form](https://github.com/final-form/react-final-form): Gérer
  l'état des formulaires
- [Redux persist](https://github.com/rt2zz/redux-persist) : Persister le state
  Redux

<SlideHeader slideTitle="Librairie native : react-native-config" />

#### [react-native-config](https://github.com/luggit/react-native-config)

- Configuration d'environnement
- Vital pour séparer votre configuration .env, .env.preprod, .env.prod
- Config aussi disponible dans le code natif

```js
import Config from "react-native-config";

Config.API_URL; // 'https://myapi.com'
Config.GOOGLE_MAPS_API_KEY; // 'abcdefgh'
```

<SlideHeader slideTitle="Librairie native : react-native-camera-kit" />

#### [react-native-camera-kit](https://github.com/teslamotors/react-native-camera-kit)

- Gestion de la caméra performante
- QR Code

```jsx
import { CameraScreen } from "react-native-camera-kit";

<CameraScreen
  actions={{ rightButtonText: "Done", leftButtonText: "Cancel" }}
  onBottomButtonPressed={(event) => this.onBottomButtonPressed(event)}
  cameraFlipImage={require("path/to/image").default.src}
  captureButtonImage={require("path/to/image").default.src}
  torchOnImage={require("path/to/image").default.src}
  torchOffImage={require("path/to/image").default.src}
  hideControls={false}
  showCapturedImageCount={false}
/>;
```
