<SlideHeader slideTitle="Installer un composant tiers" />

#### Comment trouver un bon composant ? 🔍

- Vérifier le repository Github :
  - Nombre d'étoiles
  - Nombre d'issues (rapport close/ouverte)
  - Date du dernier commit
- Jeter un oeil sur le code
