- Permet de traduire votre application

#### Librairie react-intl 👅

- Développée par **Yahoo**, elle couvre de **nombreux aspects** :
  - Gère les traductions
  - Formatage des nombres
  - Formatage des dates et des heures
  - Temps relatif ("now", "yesterday"…)
  - Formes plurielles
  - Supporte + de 150 langues

##### **React-intl** : implémentation 💻

- Le composant **IntlProvider** permet d'initialiser la librairie

```jsx
import { IntlProvider } from "react-intl";

const locale = "en";
import messagesFr from "./translations/fr.json";
import messagesEn from "./translations/en.json";

const messages = {
  fr: messagesFr,
  en: messagesEn,
};

const component = (
  <IntlProvider locale={locale} messages={messages[locale]}>
    <Chat />
  </IntlProvider>
);

ReactDOM.render(component, document.getElementById("app"));
```

##### **React-intl** : implémentation 💻

- Le composant **FormattedMessage** permet de traduire les chaines :
