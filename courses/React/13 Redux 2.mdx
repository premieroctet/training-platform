### Les middleware Redux 🤓

- Ils permettent d'ajouter sa logique **entre la phase de dispatch et la phase
  de reducing**
- Les middlewares peuvent permettre de :
  - Faire du logging
  - Remonter des crash
  - Faire des requêtes asynchrones
- Il faut utiliser la méthode **applyMiddleware()** pour en ajouter

#### Code minimal d'un middleware

```js
const customMiddleware = (store) => (next) => (action) => {
  ///////////////////////////
  // ajouter votre logique //
  ///////////////////////////

  // Passer l'action au reducer ou au middleware suivant
  next(action);
};
```
