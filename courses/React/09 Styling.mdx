#### Il existe une plusieurs manières de "styler" ses composants React 🎨

- Feuille de style
- CSS Module
- Inline
- CSS in JS

---

<SlideHeader slideTitle="Styling - Feuille de style" />

#### Feuille de style

```jsx
import React from "react";

import "./app.css";

const Button = () => {
  // You can use them as regular CSS styles
  return <div className="header" />;
};
```

---

<SlideHeader slideTitle="Styling - CSS Module" />

#### CSS Module

```css
// Button.css
.error {
  background-color: red;
}
```

```jsx
import React from "react";
import styles from "./Button.css"; // Import css modules stylesheet as styles

const Button = () => {
  // reference as a js object
  return <button className={styles.error}>Error Button</button>;
};
```

---

<SlideHeader slideTitle="Styling - Inline" />

#### Inline

```jsx
const errorButtonStyle = { backgroundColor: "red" };

const Button = () => {
  return <button style={errorButtonStyle}>Error Button</button>;
};
```

---

<SlideHeader slideTitle="Styling - CSS in JS" />

#### CSS in JS avec [styled-components](https://www.styled-components.com/)

- Utilise des composants React

```jsx
import styled from "styled-components";

const Button = styled.button`
  background: white;
  border: 1px solid red;
`;

const App = () => {
  return <Button>Save</Button>;
};
```

---

<SlideHeader slideTitle="Styling - CSS in JS" />

#### CSS in JS avec [styled-components](https://www.styled-components.com/)

- Compatible avec les props :

```jsx
import styled from "styled-components";

const Button = styled.button`
  background-color: `${props => props.color || "white"}`;
  border: 1px solid red;
`;

const App = () => {
  return <Button color="yellow">Save</Button>;
}
```

---

<SlideHeader slideTitle="Styling - CSS in JS" />

#### CSS in JS avec [styled-components](https://www.styled-components.com/)

- Héritage :

```jsx
import styled from "styled-components";

const Button = styled.button`
  background-color: `${props => props.color || "white"}`;
  border: 1px solid black;
`;

const ErrorButton = styled(Button)`
  border: 1px solid red;
`;

const App = () => {
  return <ErrorButton>Save</ErrorButton>;
}
```

---

<SlideHeader slideTitle="Styling - CSS in JS" />

#### CSS in JS avec [styled-components](https://www.styled-components.com/)

- "As" pattern :

```jsx
import styled from "styled-components";

const Button = styled.button`
  border: 1px solid red;
`;

const App = () => {
  return <Button as="a">Save</Button>; // <a href="">Save</a> et non <button></button>
};
```
