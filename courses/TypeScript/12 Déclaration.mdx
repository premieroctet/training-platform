#### Déclaration de types

- Se fait dans des fichiers `*.d.ts`
- Doit-être inclus dans votre projet (`tsconfig.json` -> `include`)

---

<SlideHeader slideTitle="Déclarations" />

#### Déclaration de types

```typescript
// types/nom-module.d.ts

declare module "nom-module" {
  const nomModule: {
    hello: (world: string) => string;
  };

  export default nomModule;
}
```
