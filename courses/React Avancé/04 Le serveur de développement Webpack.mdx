##### 🚧 **Webpack** offre un [serveur de développement](https://webpack.js.org/configuration/dev-server/) :

- ⚠️ A ne jamais utiliser en **production**
- Améliorant l'expérience de développement :
  - **Live reload** 👍️
  - **Hot reload** 👍️

---

<SlideHeader slideTitle="Serveur de développement Webpack" />

##### 🚀 **Installer le dev-server Webpack** (1/2)

- Ajouter la dépendance **dev-server** Webpack :

```
yarn add --dev webpack-dev-server
```

- Ajouter le script pour lancer le **dev-server** (dans le package.json) :

```json
"scripts": {
  "start": "webpack-dev-server --open --mode development"
},
```

---

<SlideHeader slideTitle="Serveur de développement Webpack" />

##### 🚀 **Installer le dev-server Webpack** (2/2)

- Ajouter un fichier **index.html** dans le dossier **dist** de votre projet
  pointant vers votre bundle :

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Hello</title>
    <script src="bundle.js"></script>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```

```
yarn run start
```
